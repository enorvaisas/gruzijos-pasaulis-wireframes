# Gruzijos Pasaulis Website Analysis & New Structure

## Current Website Analysis

### Current Issues:
1. **Poor Navigation Structure**: Content scattered across 80+ individual pages
2. **No Clear Hierarchy**: All pages seem equally important
3. **Mixed Content Types**: Products, events, blog posts mixed together
4. **SEO Problems**: Too many single-purpose pages, poor internal linking
5. **User Experience**: Difficult to find specific information

### Current Content Inventory:
- **Art & Culture**: Artists, exhibitions, paintings
- **Culinary Services**: Cooking classes, tasting dinners
- **Products**: Spices, textiles, accessories, gift cards
- **Events**: Private parties, corporate events
- **Information**: About us, contact, policies

## SEO Optimization Strategy

### Technical SEO:
- Clean URL structure: `/kulinariniai-mokymai/`, `/menas/menininkai/`
- Proper internal linking between related content
- Breadcrumb navigation
- Schema markup for events, products, organization

### Content SEO:
- Target keywords: "Gruzijos virtuvė", "kulinariniai mokymai Kaune", "degustacinės vakarienės"
- Local SEO: "Kaunas", "Gruzijos kultūra Lietuvoje"
- Long-tail keywords for specific services

### User Experience:
- Clear navigation hierarchy
- Search functionality
- Mobile-first design
- Fast loading times
- Clear call-to-actions
